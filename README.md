🌍 WanderLust – Airbnb Clone

A full-stack Airbnb-style web application where users can explore listings, create their own listings, upload images, view locations on maps, and leave reviews.

This project demonstrates full-stack development using Node.js, Express, MongoDB, Mapbox, and Cloudinary.

🚀 Features

🔐 User Authentication (Signup / Login / Logout)

🏠 Create, Edit, and Delete Listings

📸 Image Upload with Cloudinary

🗺️ Map Integration using Mapbox

⭐ Review and Rating System

👤 Authorization (Only owners can edit/delete listings)

📍 Geocoding to convert locations into map coordinates

💬 Flash messages for user feedback

🎨 Responsive UI using Bootstrap
